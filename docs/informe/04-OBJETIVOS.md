# II. OBJETIVOS DEL PROYECTO

---

## **2.1 OBJETIVO GENERAL**

**Desarrollar e implementar un sistema web de gestión clínica para el Servicio de Obstetricia y Neonatología del Hospital Herminda Martín**, que digitalice el proceso de registro de partos y atención neonatal, reemplazando el sistema manual basado en Excel, con enfoque en la seguridad de datos y mejora de la calidad asistencial.

---

## **2.2 OBJETIVOS ESPECÍFICOS**

### **2.2.1 DIGITALIZACIÓN DEL REGISTRO CLÍNICO**
- **Implementar formularios digitales** para el registro de datos de pacientes, control prenatal y partos
- **Reducir tiempo de registro** mediante la eliminación del proceso manual en Excel
- **Evitar la duplicación de datos** a través de un sistema centralizado
- **Establecer validaciones automáticas** para campos críticos como fechas, rangos de peso y evaluación APGAR

**Resultado Esperado:** Registro digital completo de procesos obstétricos y neonatales

---

### **2.2.2 MEJORA EN LA CALIDAD DEL REGISTRO CLÍNICO**
- **Reducir errores de registro** mediante validaciones automáticas de campos
- **Implementar validaciones en tiempo real** para datos críticos como RUT, fechas y rangos médicos
- **Establecer controles de integridad** para validar consistencia entre datos relacionados
- **Crear alertas para inconsistencias** en información médica importante

**Resultado Esperado:** Disminución significativa de errores en el registro de datos clínicos

---

### **2.2.3 AUTOMATIZACIÓN DE PROCESOS MÉDICOS**
- **Implementar evaluación APGAR estructurada** con validaciones automáticas de rangos (0-10)
- **Registrar automáticamente datos antropométricos** del recién nacido (peso, talla, circunferencia craneana)
- **Gestionar información de casos especiales** como VIH y otras condiciones médicas importantes
- **Alertar sobre valores fuera de rangos normales** para parámetros críticos

**Resultado Esperado:** Procesos médicos clave automatizados y estandarizados en el sistema

---

### **2.2.4 GENERACIÓN DE REPORTES**
- **Facilitar la generación de reportes** con datos estadísticos del servicio
- **Reducir tiempo de elaboración** de reportes mediante consultas automatizadas a la base de datos
- **Garantizar consistencia** en la información reportada
- **Exportar datos** en formatos estándar para análisis estadístico

**Resultado Esperado:** Capacidad de generar reportes estadísticos de forma eficiente y confiable

---

### **2.2.5 SEGURIDAD Y CONTROL DE ACCESO**
- **Implementar sistema de control de acceso** basado en roles para diferentes tipos de usuarios médicos
- **Asegurar protección de datos** siguiendo prácticas de seguridad para información de salud
- **Establecer auditoría de acciones** para trazabilidad de cambios en registros médicos
- **Proteger información sensible** mediante controles de acceso apropiados

**Resultado Esperado:** Sistema seguro que protege la privacidad y confidencialidad de datos médicos

---

### **2.2.6 USABILIDAD Y EXPERIENCIA DE USUARIO**
- **Diseñar interfaces intuitivas** adaptadas a los flujos de trabajo médicos
- **Optimizar para diferentes dispositivos** incluyendo tablets y computadoras de escritorio
- **Minimizar la complejidad** del sistema para facilitar su adopción
- **Lograr aceptación** del personal médico mediante diseño centrado en el usuario

**Resultado Esperado:** Interfaces fáciles de usar que mejoren la experiencia del personal médico

---

## **2.3 BENEFICIOS ESPERADOS**

#### **Matronas y Médicos Obstetras:**
- **Ahorro de tiempo:** Eliminación del registro manual en Excel
- **Reducción de errores:** Validaciones automáticas de datos
- **Mejor acceso a información:** Datos clínicos organizados y accesibles
- **Cumplimiento de estándares:** Registros completos y consistentes

#### **Pediatras y Neonatólogos:**
- **Acceso a datos del parto:** Información completa para evaluación neonatal
- **Registro estructurado:** Datos APGAR y antropométricos organizados
- **Trazabilidad:** Seguimiento completo de la atención neonatal
- **Alertas importantes:** Notificación sobre casos que requieren atención especial

#### **Personal de Enfermería:**
- **Interfaces optimizadas:** Diseño adaptado para uso en diferentes dispositivos
- **Validaciones automáticas:** Prevención de errores en el registro
- **Acceso controlado:** Permisos apropiados según rol
- **Sistema intuitivo:** Interfaz fácil de aprender y usar

### **2.3.2 BENEFICIOS PARA LA INSTITUCIÓN HOSPITALARIA**

- **Eficiencia operacional:** Reducción de tiempo en documentación manual
- **Mejora en calidad de datos:** Información más precisa y confiable
- **Cumplimiento normativo:** Mejor adherencia a estándares de documentación
- **Generación de reportes:** Capacidad mejorada para reportes institucionales
- **Indicadores de calidad:** Métricas automatizadas del servicio obstétrico
- **Optimización de recursos:** Mejor planificación basada en datos precisos

### **2.3.3 BENEFICIOS PARA PACIENTES**

- **Mayor seguridad:** Reducción de errores mediante validaciones automáticas
- **Atención más eficiente:** Personal médico dedicando más tiempo a atención directa

- **Protección de privacidad:** Controles de acceso apropiados para datos sensibles
